# grasp_binary_pro

A production-oriented binary linear heuristic using:

- greedy randomized construction (GRASP)
- feasibility repair
- one-flip local improvement
- brute-force fallback for tiny models

## Best For

- 0/1 knapsack variants
- binary packing/selection models with linear constraints

## Main Config

- `time_limit` (seconds)
- `seed`
- `iterations`
- `rcl_size`
