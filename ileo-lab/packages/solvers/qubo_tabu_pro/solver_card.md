# qubo_tabu_pro

QUBO heuristic sampler based on tabu search with aspiration and multi-start restarts.

## Best For

- MaxCut-like QUBO instances
- medium-scale binary unconstrained models

## Main Config

- `time_limit` (seconds)
- `seed`
- `iterations`
- `restarts`
- `tabu_tenure`
- `stagnation_limit`
